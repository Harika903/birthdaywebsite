# 🎂 Birthday Celebration Website
## 📌 Overview

This is an interactive and visually engaging birthday website designed to create a fun and personalized celebration experience. The website allows users to enter a name and enjoy animated effects such as balloons, sparkles, and dynamic transitions. It focuses on creativity, user interaction, and modern UI design.

# 🎮 How it Works
User enters their name on the welcome screen
The website displays a personalized birthday greeting
Animated effects like balloons and sparkles appear on the screen
User interactions (click/touch) trigger special animations
Smooth transitions enhance the overall experience
# 🛠 Technologies Used
HTML
CSS (Animations & Effects)
JavaScript (Interactivity & Events)
# 🚀 Features
Personalized greeting with user name input
Animated balloons using pure CSS (no images)
Sparkle effects triggered on user interaction
Smooth transitions and responsive UI
Attractive and colorful design for better engagement
Interactive click/touch-based animations
# ▶️ How to Run
Download or clone the project

Open the file:

index.html

Enter your name and enjoy the birthday animation 🎉
